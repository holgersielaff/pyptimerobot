### pyptimerobot

##### Config

A config may look like this:

```json
{
  "name": "MyName",

```

##### Installation
```bash
git clone https://github.com/holgersielaff/pyptimerobot
cd pytimerobot
python3 -m venv venv
./venv/bin/python3 -m pip install -r requirements.txt
```

##### Run the script

```bash
./venv/bin/python3 robot.py
```

##### Why Flask?

To build a web frontend later :)
